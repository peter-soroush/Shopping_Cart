## üõçÔ∏è About The Project

**Shopping Cart** is a robust application designed to simulate a real-world e-commerce cart experience. Whether you are building a full store or testing checkout logic, this project provides a solid foundation for managing products, calculating totals, and handling user sessions.

This project was built to demonstrate:

- Clean code architecture.
- State management for cart items.
- Dynamic price calculation (taxes, discounts, subtotal).

## üöÄ Features

Here are the key features of the application:

- [x] **Add to Cart**: Instantly add items to your basket.
- [x] **Quantity Adjustment**: Increase or decrease item counts dynamically.
- [x] **Remove Items**: Delete products from the cart with a single click.
- [x] **Real-time Totals**: Automatic calculation of Subtotal, Tax, and Grand Total.
- [ ] **Promo Codes**: Support for discount coupons. _(Future Update)_
- [ ] **Responsive Design**: Works on Mobile, Tablet, and Desktop.

## üèÅ Getting Started

Follow these steps to get a local copy up and running.

### Prerequisites

- Make sure you have **[Node.js / Python / Java]** installed on your machine.
- Package manager: `npm` / `pip` / `maven`.

### Installation

1.  **Clone the repository**
    ```sh
    git clone [https://github.com/peter-soroush/Shopping_Cart.git](https://github.com/peter-soroush/Shopping_Cart.git)
    ```
2.  **Navigate to the project directory**
    ```sh
    cd Shopping_Cart
    ```
3.  **Install dependencies**

    ```sh
    # Example for JS
    npm install

    # Example for Python
    # pip install -r requirements.txt
    ```

4.  **Run the application**
    ```sh
    npm start
    # or
    python main.py
    ```

## üñ•Ô∏è Usage

Once the application is running, navigate to `http://localhost:3000` (or your specific port) in your browser.

1.  Browse the product list.
2.  Click **"Add to Cart"** on any item.
3.  Open the Cart view to see your selected items.
4.  Proceed to "Checkout" to see the final calculation.

> **Note:** If you encounter any issues, check the console logs for detailed error messages.
